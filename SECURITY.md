# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Email **ian@allowayllc.com** with a description of the issue
3. Include steps to reproduce if possible

We will acknowledge receipt within 48 hours and coordinate a fix timeline with you.

## Scope

This repository is primarily a documentation/rubric project. Security concerns relevant to this repo include:

- Malicious edits to the checklist that could mislead evaluators
- Credential leakage in examples or linked content

For security issues in the browser agents *being evaluated* by this rubric, contact those projects directly.
